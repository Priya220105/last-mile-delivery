═══════════════════════════════════════════════════════════════════════════════
 🎯 CORE FILES TO UNDERSTAND
═══════════════════════════════════════════════════════════════════════════════

1. RATE CALCULATION (Most Important!)
   File: backend/helpers/rateCalculator.js
   
   Main Function: calculateCharge(order, db)
   ├─ Gets zones from pincode mapping
   ├─ Calculates volumetric weight
   ├─ Gets billable weight
   ├─ Maps to weight slab
   ├─ Fetches rate card from DB
   ├─ Looks up base charge
   ├─ Calculates COD surcharge
   └─ Returns complete breakdown

2. ORDER CREATION
   File: backend/routes/orders.js
   
   Function: POST /orders
   ├─ Validates input
   ├─ Calls calculateCharge()
   ├─ Creates order in DB
   ├─ Returns order ID
   └─ Customer sees charges first

3. ADMIN RATE MANAGEMENT
   File: backend/routes/admin.js
   
   Function: POST /rate-cards
   ├─ Updates B2C rates
   ├─ Updates B2B rates
   ├─ Sets COD surcharge
   ├─ Stores in DB
   └─ New orders use updated rates immediately

4. TRACKING TIMELINE
   File: backend/routes/orders.js
   
   Pattern: Event Sourcing
   ├─ Each status change appends to array
   ├─ Never delete/modify past events
   ├─ Stores timestamp and actor
   └─ Complete audit trail

5. FRONTEND ORDER FORM
   File: frontend/src/pages/CustomerDashboard.jsx
   
   Process:
   ├─ User fills form
   ├─ Clicks "Calculate Charge"
   ├─ API returns breakdown
   ├─ Shows ₹ amount
   ├─ User clicks "Confirm"
   └─ Order is created
